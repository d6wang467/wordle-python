A simple command-line version of the popular Wordle game built in Python.

Players have 6 attempts to guess a randomly selected 5-letter word.

After the game:
- If you win: you receive a random reward
- If you lose: you receive a random challenge

Features
- 6 attempts per game
- Input validation (must be a real 5-letter English word)
- Colored terminal feedback:
  - Green: correct letter, correct position
  - Yellow: correct letter, wrong position
- Random reward or challenge system
